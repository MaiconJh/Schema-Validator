Installation

Install and bootstrap Schema Validator with project-friendly defaults.

Install flow

  1. Add plugin artifact to your environment.
  2. Start once to generate default files.
  3. Confirm schema directories exist.
  4. Restart after initial edits.

Verify installation

# check runtime logs for plugin load status
# confirm schema directory paths are present

Troubleshooting

Plugin does not load

  • Confirm version compatibility.
  • Inspect server logs for dependency issues.

Schemas are not discovered

  • Confirm schema path settings.
  • Verify file permissions and extensions.
