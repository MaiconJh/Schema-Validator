Getting started

Understand prerequisites, concepts, and terminology before installation.

Core concepts

  • Schema registration: loads and resolves schemas.
  • Validation dispatcher: routes data nodes to validators.
  • Integration layer: exposes commands and results to Skript.

Prerequisites

  • Compatible Java runtime.
  • Access to plugin configuration and schema files.
  • Familiarity with JSON/YAML structure.

Terminology

Schema

A rule set defining allowed data patterns.

Data file

A JSON/YAML input validated against a schema.

Validation result

A success/failure status with error details.
